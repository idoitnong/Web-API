## 센서(데이터)관리

| METHOD | URL                 | 설명             |
| ------ | ------------------- | ---------------- |
| GET    | /sensors            | 센서 목록        |
| POST   | /sensors            | 센서 추가        |
| GET    | /sensors/:sensor_id |                  |
| POST   | /sensors/:sensor_id | 센서 데이터 추가 |
| PUT    | /sensors/:sensor_id | 센서 정보 주ㅅ   |
|        |                     |                  |
